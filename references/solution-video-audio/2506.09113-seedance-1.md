Seedance 1.0 论文核心解读总结
Seedance 1.0 是字节跳动 Seed 团队研发的高性能、推理高效的视频生成基础模型，核心解决了当前视频生成模型难以同时平衡提示词跟随性、运动合理性、视觉质量的行业痛点，原生支持中英双语、文本到视频（T2V）/ 图像到视频（I2V）多任务统一、多镜头生成，2025 年 6 月已落地火山引擎 α、豆包、剪映即梦等平台，在 2025 年 6 月 10 日 Artificial Analysis 的 T2V 和 I2V 专业榜单中均位列第一。
一、核心技术四大核心改进
模型的核心能力源于四大技术体系的融合优化，也是其与竞品的核心差异点：
多源数据精修 + 精准视频字幕：构建大规模高质量多源视频数据集，通过多阶段精修保证数据多样性与质量，搭配精准的双语视频字幕系统，实现复杂场景的全面学习。
高效架构设计 + 专属训练范式：解耦时空层的架构设计，让模型原生支持多镜头生成，且能在单一模型中联合学习 T2V 和 I2V 任务。
精细化后训练优化：结合细粒度监督微调（SFT）和视频专属多维度奖励 RLHF，全面提升运动自然度、结构一致性和视觉保真度。
极致推理加速：通过多阶段蒸馏策略 + 系统级优化实现端到端 10× 推理加速，在 NVIDIA-L20 上生成 5 秒 1080p 分辨率视频仅需 41.4 秒，无性能损失。
二、核心模型设计
模型采用模块化架构设计，各模块协同实现高质量、高可控的视频生成，核心包含四大模块：

2. 扩散 Transformer（DiT）
模型的核心扩散骨干，是多任务统一和多镜头生成的关键：
解耦时空层：空间层处理单帧内注意力聚合，时间层处理帧间注意力计算，提升训练 / 推理效率；
采用 MMDiT 架构：空间层融合视觉 + 文本令牌，时间层仅处理视觉令牌，为两种模态设计独立权重；
多镜头 MM-RoPE：在 3D RoPE 基础上为文本令牌增加 1D 位置编码，支持视觉 / 文本令牌交错序列，适配多镜头视频训练；
统一任务公式：通过通道维度拼接噪声输入与参考帧、二进制掩码标记指令帧，实现 T2V、I2V、文本到图像的任务统一。
3. 扩散精炼器（Diffusion Refiner）
采用级联扩散框架实现高分辨率视频生成：先由基础模型生成 480p 低分辨率视频，再通过精炼器上采样至 720p/1080p，同时增强视觉细节和纹理；精炼器由预训练基础模型初始化，以低分辨率视频为条件训练。
4. 提示工程（PE）
基于Qwen2.5-14B构建，将用户自然语言提示词转换为符合 DiT 训练的密集视频字幕格式：
先通过 SFT 阶段学习基础改写能力，区分 T2V/I2V 不同的提示词风格；
再通过 Direct Preference Optimization（DPO）做强化学习，解决模型幻觉问题，保证改写结果的语义一致性。
三、高质量视频数据处理流程
模型性能高度依赖训练数据的规模、多样性和质量，团队设计了三阶段自动化数据处理流水线，将异构原始视频转化为精细化训练数据集：
多元化数据采集：从合规的公共 / 授权库获取数据，覆盖时长、分辨率、主体、场景、风格等多维度，最大化数据多样性；
多阶段数据精修：依次完成镜头感知时间分割、视觉叠加修正（去除 logo / 字幕）、质量安全过滤、语义去重、分布重平衡（解决类别失衡），仅保留符合高标椎的视频片段；
视频字幕与编码：训练基于 Tarsier2 的双语密集视频字幕模型（融合动静特征描述），同时完成 VAE 编码生成视觉嵌入，为模型训练提供文本 + 视觉特征。
此外，搭建了三层高效工程基础设施（统一平台层 + 计算框架层 + 底层资源层），基于 BMF、Ray 实现 CPU/GPU/NPU 异构计算，支持海量视频数据的高吞吐处理。
四、分阶段模型训练流程
基础模型和扩散精炼器均遵循 ** 预训练→持续训练（CT）→监督微调（SFT）→人类反馈对齐（RLHF）** 的四阶段训练流程，各阶段层层优化模型性能：
1. 预训练
采用流匹配框架 + 速度预测，结合分辨率感知的时间步变换，为高分辨率 / 长时长视频增加噪声扰动；
渐进式训练：从 256px 文本到图像训练开始，逐步引入更高分辨率（256px→640px）、更高帧率（12fps→24fps）的视频模态，同时保留 20% 的 I2V 任务占比。
2. 持续训练（CT）
针对预训练中 I2V 任务占比低的问题优化：将 I2V 任务占比提升至 40%，精选高美学、高运动动态的训练数据，设计长 / 短双字幕（长字幕含动静特征，短字幕仅聚焦运动），提升模型的 I2V 能力和 T2V 的视觉保真度。
3. 监督微调（SFT）
人工精选覆盖多风格 / 多运动类型的高质量视频 - 文本对，保证数据分布平衡；
分模型训练后进行模型融合，采用小学习率 + 早停策略，防止过拟合并保持文本可控性，提升视觉质量和运动一致性。
4. 人类反馈对齐（RLHF）
视频领域专属的 RLHF 设计，是模型贴近人类偏好的关键：
构建多维度反馈数据：采集训练集 / 用户提示词，搭配模型不同阶段生成的视频对，进行人类偏好标注；
设计三大专属奖励模型：基础奖励模型（提升图文对齐、结构稳定性）、运动奖励模型（减少伪影、增强运动生动性）、美学奖励模型（基于视频关键帧优化视觉美学）；
优化策略：直接最大化多奖励模型的复合奖励，在基础模型和精炼器上均实现 RLHF，且通过扩散模型与奖励模型的多轮迭代学习，提升性能上限。
五、推理与训练基础设施优化
为实现高效推理 + 大规模训练，团队从模型层和系统层做了全方位的基础设施优化，兼顾速度、性能和硬件利用率。
1. 推理优化（端到端 10× 加速）
模型加速：DiT 采用 TSCD 蒸馏 + Score Distillation + 对抗训练实现 4× 加速，VAE 解码器通过通道瘦身实现 2× 加速，无视觉质量损失；
推理基础设施：核融合（吞吐量提升 15%）、细粒度混合精度量化 + 稀疏化、定制化混合并行策略（通信开销降至 Ulysses 的 1/4）、异步卸载策略（内存受限设备性能损失 < 2%）、VAE 分布式混合并行、流水线优化，全面提升推理吞吐量。
2. 训练基础设施优化
支持数千 GPU 的大规模长上下文视频模型训练，分预训练和后训练针对性优化：
预训练：高性能核融合（减少 90% 全局内存流量）、HSDP+Ulysses 序列并行、多级激活检查点（MLAC）、运行时工作负载平衡、多级容错（定期检查点 + 节点健康检测 + 元张量初始化），保证训练的稳定性和高吞吐；
后训练：针对内存竞争、复杂训练模式、多样工作负载的问题，设计动态内存管理框架（CPU 卸载 + 重计算），适配预训练的并行策略，优化内存碎片和参数释放问题，提升内存利用率。
六、模型性能表现
Seedance 1.0 在外部公共评测和内部专业评测中均表现出 SOTA 水平，且实现了单模型在 T2V/I2V 双任务的同时领先，打破了竞品 “单任务优、另一任务弱” 的现状。
1. 外部评测：Artificial Analysis Arena
T2V 和 I2V 榜单Elo 评分均位列第一，其中 I2V 任务领先第二名 Veo 3 超 100 分；
参与评测的竞品包括 Veo 3、Kling 2.0、Runway Gen4、OpenAI Sora、Wan 2.1 等，Seedance 1.0 在用户偏好上表现出显著优势。
2. 内部评测：自研 SeedVideoBench 1.0
构建专业评测基准：包含 T2V/I2V 各 300 个提示词，覆盖特效、电商、PGC 等真实场景，联合电影导演制定专业评测维度；
核心评测维度：运动质量（结构准确性、运动合理性、稳定性）、提示词跟随性（动作 / 主体 / 风格的一致性）、美学质量（纹理、光影、风格）、I2V 专属的原图保留性（主体 / 风格 / 色彩的一致性）；
评测结果：在绝对评分（5 分制）和GSB 两两对比中均领先 Kling 2.1、Veo 3、Wan 2.1、Sora、Runway Gen4 等竞品，其中提示词跟随性是核心优势，运动质量和视觉质量与竞品持平或更优。
七、特色能力
多镜头生成能力：原生支持连贯的多镜头叙事，镜头切换流畅，主体在时空变换中保持一致性，支持电影级镜头语言（正反打、插入镜头、匹配剪辑、动作剪辑），能处理复杂的叙事性提示词；
多风格对齐能力：泛化性强，支持多种视觉风格（黑白默片、港风、赛博朋克、水墨动画、像素风等），T2V 可直接生成细粒度风格视频，I2V 能精准保留参考图像的视觉特征，无需任务专属调优。
八、核心优势与落地
1. 核心竞争优势
多任务统一：单模型支持 T2V/I2V / 多镜头生成，原生中英双语，无需多模型拼接；
性能均衡：首次实现提示词跟随性、运动合理性、视觉质量的三重平衡；
推理高效：10× 端到端加速，生成速度远超商用竞品，推理成本显著降低；
叙事能力强：原生支持电影级多镜头叙事，填补了现有模型的叙事短板；
风格泛化好：覆盖多类专业视觉风格，适配专业创作和日常创意场景。
